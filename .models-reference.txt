  КАК ИСПОЛЬЗОВАТЬ
═══════════════════════════════════════════════════════════

1. Откройте: lib/model-config.ts

2. Найдите активную конфигурацию:
   export const MODEL_CONFIGS = {
     'primary': {
       name: 'ЗДЕСЬ_ВСТАВИТЬ', // ← вставьте имя из списка выше
       ...
     }
   }

3. Скопируйте нужное имя модели отсюда

4. Вставьте в name: '...'

5. Перезапустите: npm run dev
